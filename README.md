## How to run
Just open the terminal, move to root directory and write the command bellow
```sh
npm run build
```

## How to see site
Just open the terminal, move to root directory and write the command bellow
```sh
npm run dev
```

P.s.
Dont foget to install [Node.js](https://nodejs.org/en)
